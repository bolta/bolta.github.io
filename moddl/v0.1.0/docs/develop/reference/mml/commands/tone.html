MML command c, d, e, f, g, a, b

音符を記述します。

書式

(c|d|e|f|g|a|b)[+[+…]|-[-…]][length][&]

詳細

音名は、c, d, e, f, g, a, b がそれぞれド、レ、ミ、ファ、ソ、ラ、シに対応します。

音名の後に + をつけるとシャープ、- をつけるとフラットを表します。同じ種類の記号であれば複数つけることもできます。

length は音符の長さを表します。省略すると l コマンドで設定した値が使われます。

最後に & をつけるとスラーになり、この音符のノートオフと、直後の音符のノートオンが省略されます。
