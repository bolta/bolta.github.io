MML command l, L

デフォルト音長を設定します。

書式

(l|L)<length>

パラメータ

名前

型

必須/省略時

説明

length

Number

必須

音長を指定します。

詳細

音符・休符の音長を省略した場合に割り当たる音長を設定します。

length に指定できるのは 0 より大きい整数値のみです。付点やタイを使うことはできません。

l コマンドを一度も使っていない状態では、l4 が指定されているものと見なされます。

大文字の L を使うこともできます（意味は同じです）。
