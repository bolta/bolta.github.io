MML command q

音長比（gate rate）を設定します。

書式

q<value>

パラメータ

名前

型

必須/省略時

説明

value

Number

必須

音長比を指定します。

詳細

「音符の音長」に対する「実際に発音する音長」の比率を設定します。

value の最小は 0、最大は 8 で、実際に発音する音長は音符の音長の value/8 倍になります。

音長は tick 単位で計算され、端数は切り捨てられます。

q コマンドを一度も使っていない状態では、q8 が指定されているものと見なされます。
