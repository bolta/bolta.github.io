コマンドライン

コマンドラインの書式は以下の通りです：

moddl [<options>] <path/to/source.moddl>

<options> には以下のものを指定することができます：

長いオプション

短いオプション

意味

--output=$output

-O$output

出力方法を指定します。--output-file とは択一です。
$output には audio, stdout, null のいずれかを指定します。
audio を指定すると、オーディオで出力されます。
stdout を指定すると、標準出力に各サンプルの数値が出力されます。
null を指定すると、何も出力されません。
デフォルトは audio です。

--output-file=path/to/output.wav

-opath/to/output.wav

出力先として wav ファイルを指定します。また、出力先のファイルパスを指定します。--output とは択一です。
既存ファイルは上書きされます。
wav ファイルの品質は 44.1 kHz, 16 ビットに固定されています。チャンネル数はソースファイルの記述に依存して 1 または 2 になります。

--stack-size=$size

-S$size

ModDL/MML ソースコードを解釈する際のスタックサイズを指定します。
複雑な式・長い式を記述してエラーが発生した場合、このオプションで大きな値を指定すると解決する場合があります。
$size はバイト単位の整数値で指定します。整数値の後に k（K）または m（M）をつけることで KiB・MiB 単位で指定することもできます。
デフォルトは 10m です。

その他

  • ファイルパスの区切り文字には \ と / の両方が使えます。
