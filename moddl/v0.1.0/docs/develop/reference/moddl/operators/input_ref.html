operator = =>

入力参照（モジュールへの入力を明示的に参照するための変数）を定義します。

記法

=input=> moduleDef

引数

名前

型

必須/省略時

説明

input

(Identifier)

必須

入力参照を設定する変数名（式ではありません）。

moduleDef

ModuleDef

必須

モジュール定義。定義中に input を用いることができます。

値

型

説明

ModuleDef

moduleDef の値。

使用例

以下の例では、2 つのオシレータを、周波数を少しずらして発音させるモジュールを定義します：

let detuneCent = 10;
=freq=> freq | sineOsc + (2 ^ (detuneCent / 1200) * freq) | sineOsc

詳細

変数 input に設定される値は、ModuleDef 型の値です。

使い方の詳細は「モジュールの構築」を参照してください。
