/ リファレンス / ModDL リファレンス / 演算子 / @

operator @

レジスタを定義します。

記法

value@label

引数

名前

型

必須/省略時

説明

value

Number | ModuleDef

必須

レジスタを定義する対象。

label

(Label)

必須

レジスタの名前となるラベル（式ではありません）。

値

型

説明

ModuleDef

レジスタとしてラベルが付与された ModuleDef。

詳細

value が Number の場合、数値を保持するノードの ModuleDef に変換された上でラベルが付与されます。

value が ModuleDef の場合はそのままラベルが付与されます。

label に指定できるのは識別子（たとえば foo123）か、それを . でつないだもの（たとえば foo123.bar456.baz789）です。
