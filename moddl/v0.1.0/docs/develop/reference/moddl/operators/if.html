operator if

条件分岐を行います。

記法

if cond then thenValue else elseValue

型定義

名前

型定義

ThenValue

*

ElseValue

*

引数

名前

型

必須/省略時

説明

cond

Number | ModuleDef

必須

条件。真偽値として解釈されます。

thenValue

ThenValue

必須

cond が truthy である場合の値。

elseValue

ElseValue

必須

cond が falsy である場合の値。

値

型

説明

ThenValue | ElseValue

cond の値に応じた、thenValue か elseValue のいずれかの値。

詳細

else 以降は式が一続きに解釈できる限り、全て elseValue として解釈されます。このため、望む結果を得るために括弧が必要になる場合があります：

// sineOsc か triangleOsc * 2 が得られる
if cond then sineOsc else triangleOsc * 2

// sineOsc * 2 か triangleOsc * 2 が得られる
(if cond then sineOsc else triangleOsc) * 2

この演算子は Number と ModuleDef に対してオーバーロードされています。詳細は「オーバーロードされた演算」を参照してください。
