詳細

引数が Number または ModuleDef であることを要求しますが、それ以外の作用はありません。

単項 - 演算子との対称性のために設けられています。

この演算子は Number と ModuleDef に対してオーバーロードされています。詳細は「オーバーロードされた演算」を参照してください。
