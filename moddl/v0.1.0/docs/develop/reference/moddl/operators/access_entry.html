operator .

連想のキーに対応する値を取得します。

記法

assoc.key

型定義

名前

型定義

Value

*

引数

名前

型

必須/省略時

説明

assoc

{ [key]: Value }

必須

連想の値。

key

(Identifier)

必須

キーの名前（式ではありません）。

値

型

説明

Value

キーに対応する値。

使用例

@let :assoc, {
  foo: 42,
  bar: {
    baz: [0, 10, 20, 30],
  } \
}

assoc.foo
// ==> 42

assoc.bar.baz~at(2)
// ==> 20

assoc.qux
// error: EntryNotFound

詳細

key は識別子でなければなりません（任意の文字列を指定することはできません）。

また、key は assoc に存在するものでなければなりません。存在しない場合 EntryNotFound エラーが発生します。
