詳細

関数を任意の数・任意の型の引数に対して適用し、結果の値を得ます。

引数の数は 0 個以上の任意の数が可能です。

ModDL の関数は副作用をもたないため、0 引数の関数は基本的に定義する意味がありませんが、関数にデフォルト引数が定義されている場合、適用時に明示する引数がなくなることはあります。

糖衣構文として ~ ( ) があり、func(arg1, arg2, ...) は arg1~func(arg2, ...) と等価です。
