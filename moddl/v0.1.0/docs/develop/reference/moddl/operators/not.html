operator !

数値を真偽値として解釈した上で、その真偽を反転し、Boolean として返します。

記法

! arg

型定義

名前

型定義

(Arg, Result)

(Number, Boolean)
| (ModuleDef, ModuleDef)

引数

名前

型

必須/省略時

説明

arg

Arg

必須

項。

値

型

説明

Result

式の値。

使用例

[2, 1, 0, -1, -2]~map(x -> ! x)
// ==> [-1, -1, 1, 1, 1]

// Number から真偽値が等しい Boolean への変換
[2, 1, 0, -1, -2]~map(x -> !! x)
// ==> [1, 1, -1, -1, -1]

詳細

この演算子は Number と ModuleDef に対してオーバーロードされています。詳細は「オーバーロードされた演算」を参照してください。
