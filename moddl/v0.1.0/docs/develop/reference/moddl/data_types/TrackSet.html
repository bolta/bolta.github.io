data type TrackSet

トラックセットを表す型です。トラックセットは、1 つ以上のトラック名の集合です。

値の記法

^ に続いてトラック名（英数字 1 文字）を羅列して表します。

例を以下に示します：

^a
^678bcDE

それぞれ「トラック a」、「トラック 6, 7, 8, b, c, D, E」を表します。

詳細

トラックセットは（その名前に反して）すでに存在するトラックの集合ではなく、単なるトラック名の集合です。

したがって、トラックセットに含まれるトラック名をもつトラックがすでに存在する必要はなく、トラックセットは新たなトラックを定義する際と、すでに定義されたトラックを参照する際の両方で用いられます。

例として、effect 構築文は 2 つのトラックセット tracks, sources を引数に取りますが、tracks はこれから定義するトラック、sources はその入力となるトラックを表します。したがって tracks ではまだ存在しないトラック名、sources ではすでに存在するトラック名を指定しなければなりません。

^ 記号の由来は正規表現です。

「トラック名の集合」は、MML 文であれば先頭に書かれるものであることから、正規表現で「先頭」を表す ^ を付ける仕様になりました。
