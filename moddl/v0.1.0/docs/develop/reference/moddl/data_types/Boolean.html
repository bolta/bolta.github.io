data type Boolean

Number 型の部分集合であり、真偽値を表す必要最小限の値だけを含む型です。

正式な型ではなく、仕様を記述する目的にのみ使われます。

記法

基本的に定数である true, false を使います。それらの値である 1 または -1 を直接記述することもできます。

記述例

true
false

詳細

Number 型の項でも述べた通り、真偽値のための独立した型はありませんが、便宜上、1 と -1 の 2 つの数値のみから成る集合を Boolean 型と呼ぶ場合があります。

たとえば関数の適用結果が Boolean 型であると言った場合、その関数は 1 か -1 だけを返し、その他の数値や数値以外の値は返さないことを意味します。

Boolean 型は関数などの仕様を記述するための便宜的な概念であり、ModDL の型システム上に正式に存在する型ではありません。たとえば type 関数が "Boolean" を返すことはありません：

[true, false]~map(type)
// ==> ["Number", "Number"]
