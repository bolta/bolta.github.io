data type NodeDef

ノード定義を表す型です。

ノード定義とはモジュール定義の特別な場合であり、そこから生成されるモジュールが単体のノードであるものです。

ノード定義以外のモジュール定義と異なり、*** document not found 演算子を使ってパラメータ入力を指定することができます。

NodeDef 型は ModuleDef 型の部分型です。

値の記法

組み込みライブラリに収録されたノード定義は、全て NodeDef 型の値です。

これ以外のノード定義を新規に作り出す方法は、現時点ではありません。

NodeDef 型の値に *** document not found 演算子を適用した結果は、（ModuleDef 型の値ではありますが）NodeDef 型の値ではありません。
