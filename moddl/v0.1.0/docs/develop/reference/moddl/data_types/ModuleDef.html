data type ModuleDef

モジュール定義を表す型です。

モジュール定義とは、モジュールを構成するノードとその接続関係を ModDL の式として記述したものです。

NodeDef 型は ModuleDef 型の部分型です。

値の記法

モジュールの構造の多彩さを反映して、モジュール定義の記法も多岐にわたっています。たとえば以下のようなものはモジュール定義です：

  • ノード定義（NodeDef 型の値）は、そのままモジュール定義でもあります。
  • ノード定義にパラメータ入力を接続したものはモジュール定義です（ノード定義ではありません）。
  • 数値またはモジュール定義を | 演算子で接続したものはモジュール定義です。
  • モジュール定義に対して数値演算（演算子または関数）を施した結果はモジュール定義です。
  • 数値を @ 演算子でレジスタに設定したものはモジュール定義です。

それぞれの例を以下に示します：

// ノード定義（ノード定義であり、モジュール定義でもある）
adsrEnv

// ノード定義 + パラメータ入力（ノード定義ではない）
adsrEnv { attack: 0.1, release: 0.5 }

// 数値またはモジュール定義を接続
440 | sineOsc
pulseOsc | lpf { 3000, 10 }

// モジュール定義による数値演算
triangleOsc * 2
sin(phase())

// 数値をレジスタ化
4000@cutoff
