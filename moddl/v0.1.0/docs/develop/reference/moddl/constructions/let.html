construction let

トップレベル変数を定義します。

パラメータ

名前

型

必須/省略時

説明

name

QuotedIdentifier

必須

定義する変数の名前を指定します。

value

any

必須

変数の値を指定します。

詳細

let 文で定義した変数は、同じ ModDL ファイル内の、定義個所より後の全ての式から参照することができます。

let 文で定義した変数は、他の ModDL ファイルから参照することはできません。変数定義を ModDL ファイル間で共有するには export 文と import 関数を使います。

@let :cutoff, 1000
@instrument ^a, sawOsc | lpf { freq: cutoff, q: 10 } * adsrEnv
