construction ticksPerBar

1 小節（4 拍）あたりの tick 数を指定します。

パラメータ

名前

型

必須/省略時

説明

ticks

Number

必須

1 小節（4 拍）あたりの tick 数を指定します。

詳細

ticksPerBar 文がない場合、1 小節（4 拍）あたり 384 ticks となります。

糖衣構文として ticksPerBeat 文があります。

@ticksPerBar 20
@instrument ^a, ...
a c4 // OK
a c5 // OK
a c8 // エラー
a c4. // エラー（c4^8 と等しいため）
