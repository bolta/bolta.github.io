construction solo

ソロ演奏するトラックを指定します。

パラメータ

名前

型

必須/省略時

説明

tracks

TrackSet

必須

ソロ演奏するトラックを指定します。

詳細

指定したトラック以外の音が鳴らなくなります。ModDL ファイル内のどこに記述するかに依らず、演奏全体に対して有効です。

mute 文と solo 文は互いに排他的であり、一方だけが有効です。また複数の mute/solo 文がある場合、最後の 1 つだけが有効です。

メインファイル以外では無視されます。

@instrument ^abcde, ...
a ...
b ...
c ...
d ...
e ...

@solo ^abc

トラック a, b, c だけが演奏されます。
