construction instrument

インストゥルメントトラックを定義します。

パラメータ

名前

型

必須/省略時

説明

tracks

TrackSet

必須

定義する対象のトラックを指定します。

複数トラックを指定した場合、その各々のトラックに対して同じ構成のインストゥルメントが生成されます。

instrument

ModuleDef

必須

トラックに割り当てるインストゥルメントを定義します。

詳細

@instrument ^ab, pulseOsc { duty: 0.25 } | lpf { 3000 }* adsrEnv { initial: 0.5, attack: 0.1, decay: 1, sustain: 0.3, release: 0.3 }

トラック a, b に、pulseOsc, lpf, adsrEnv で構成されるインストゥルメントを設定します。
