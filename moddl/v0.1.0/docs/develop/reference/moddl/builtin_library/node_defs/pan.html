node def pan

パンポットです。入力信号を左右方向の所定の位置に配置したステレオ信号を出力します。

主入力

入力信号です。必ずモノラル信号でなければなりません。

パラメータ入力

名前 必須/省略時 説明
pos 必須 音像の位置です。-1 で最も左、0 で中央、1 で最も右となります。

イベント

イベントを受け取りません。

出力

入力信号を pos に応じた左右方向の位置に配置したステレオ信号です。

使用例

# トラック a の定位を少し右へ
@instrument ^a, (sineOsc * adsrEnv) | pan { pos: 0.1 }

詳細

pos の値と出力の左右チャンネルの振幅との関係（pan law）は、AMEI で提唱された以下の仕様に基づいています（ただし、AMEI では MIDI の仕様に合わせて最も左を 0 または 1、中央を 64、最も右を 127 としています）：

https://amei.or.jp/midistandardcommittee/Recommended_Practice/e/rp36.pdf

モノラル信号とステレオ信号をミックスすると、モノラル信号は左右チャンネルに複製されてステレオ信号に拡張されますが、そうして拡張された結果のステレオ信号と、モノラル信号に pan { pos: 0 } を適用した結果のステレオ信号は異なることに注意してください。

pan { pos: 0 } では出力信号の振幅は入力信号の cos(tau / 8) = sin(tau / 8) ≒ 0.7071 倍になります。複製の場合は振幅は変わらないので、pan を適用した方が振幅が小さくなります。

意図しない結果を避けるため、pan を適用するトラックと適用しないトラックは混在させず、適用したいトラックが 1 つでもある場合は、全てのトラックに適用することをおすすめします。

主入力は必ずモノラル信号でなければなりません。ステレオ信号の場合は moddl がクラッシュします。

クラッシュするのは不具合であり今後修正される予定ですが、将来において pan がステレオ信号を受け取れるようになるかどうかは未定です（エラーになるなどの可能性もあります）。
