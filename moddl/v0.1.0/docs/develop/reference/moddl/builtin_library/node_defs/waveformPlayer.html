node def waveformPlayer()

波形を再生するノードを提供します。

関数の引数

名前

型

必須/省略時

説明

waveform

Waveform

必須

初期状態で再生する波形。

主入力

周波数を受け取ります。

パラメータ入力

入力はありません。

イベント

種別

キー

説明

Note

ノートオンで波形を startOffset から再生します。ノートオフは無視します。

Set

waveform

値を Waveform 型の値として解釈し、再生する波形を指定された波形に切り替えます。

出力

波形を再生します。

使用例

@waveform :wave1, "./path/to/wave1.wav"

@instrument ^a, waveformPlayer(wave1) * adsrEnv

詳細

waveformPlayer はノード定義を提供する関数であり、それ自体はノード定義ではありません。ノード定義を利用するには関数適用の ( ) 演算子が必要です。
