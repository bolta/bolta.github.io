function containsKey

連想に指定したキーが含まれるかどうか判定します。

引数

名前

型

必須/省略時

説明

assoc

Assoc

必須

任意の連想。

key

String

必須

キー名。

値

型

説明

Boolean

assoc がキー key を持つかどうか。

使用例

{ foo: 42, bar: [0, 1, 2], baz: { hoge: 99 } }~containsKey("bar")
// ==> true

{ foo: 42, bar: [0, 1, 2], baz: { hoge: 99 } }~containsKey("hoge")
// ==> false
