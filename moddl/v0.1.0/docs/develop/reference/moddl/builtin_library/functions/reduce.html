function reduce

配列の各要素に対して関数による集計を行い、集計結果の値を得ます。

型定義

名前

型定義

Elem

*

Result

*

引数

名前

型

必須/省略時

説明

source

Elem[]

必須

集計元の配列。

initial

Result

必須

集計結果の初期値。

reducer

(Result, Elem) -> Result

必須

集計を行う関数。

現時点までの結果と新しい要素を受け取り、新しい結果を返します。

値

型

説明

Result

集計結果の値。

使用例

// 数値の配列から、絶対値が最大の数値を求める（空の配列では 0）
let maxByAbs = array -> array~reduce(0,
    (maxSoFar, elem) -> if abs(elem) > abs(maxSoFar) then elem else maxSoFar);
[-10, 0, 5, 99, -800, 42]~maxByAbs()
// ==> -800
