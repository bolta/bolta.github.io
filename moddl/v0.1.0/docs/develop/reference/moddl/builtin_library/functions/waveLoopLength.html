function waveLoopLength

波形を表現した配列を受け取り、波形のループの長さ（サンプル数）を返します。

引数

名前

型

必須/省略時

説明

waveformData

(Number | Number[])[]

必須

波形データを表す配列。

値

型

説明

Number

波形データのループのサンプル数。

使用例

[0, 1, 2, 3, 4, [5, 6, 7, 8]]~waveLoopLength()
// ==> 4

[0, 1, 2, 3, 4, 5, 6, 7, 8]~waveLoopLength()
// error: IndexOutOfBounds

詳細

波形にループがない場合は IndexOutOfBounds エラーが発生します。

波形にループがあることが保証されない場合は、事前に waveIsLooped 関数でチェックしてください。
