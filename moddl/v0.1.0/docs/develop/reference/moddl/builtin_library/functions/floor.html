function floor

与えられた数値を上回らない最大の整数を求めます。

arg を上回らない最大の整数。

詳細

floor と trunc は、arg >= 0 のときは同じ結果を返しますが、 arg < 0 のときは異なる結果を返します。たとえば floor(-3.5) == -4 に対して trunc(-3.5) == -3 となります。

この関数は Number と ModuleDef に対してオーバーロードされています。詳細は「オーバーロードされた演算」を参照してください。
