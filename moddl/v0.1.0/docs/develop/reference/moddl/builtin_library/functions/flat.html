function flat

2 次元の配列（配列の配列）を、内側の配列を全て連結した 1 次元の配列に変換します。

型定義

名前

型定義

Elem

*

引数

名前

型

必須/省略時

説明

arrays

Elem[][]

必須

配列の配列。

値

型

説明

Elem[]

arrays に含まれる全ての配列を連結した配列。

使用例

[[0, 1], [2, 3, 4], [5, 6, 7]]~flat()
// ==> [0, 1, 2, 3, 4, 5, 6, 7]

詳細

この関数は配列の入れ子を 1 段階解消しますが、それ以上は解消しません。

たとえば [[0, 1], [2, [3, 4], 5], [6, 7]]~flat() の結果は [0, 1, 2, [3, 4], 5, 6, 7] であり、[0, 1, 2, 3, 4, 5, 6, 7] ではありません。
