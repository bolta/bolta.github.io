function filter

配列の各要素のうち、関数で表された条件を満たすものだけを含む新たな配列を得ます。

型定義

名前

型定義

Elem

*

引数

名前

型

必須/省略時

説明

source

Elem[]

必須

変換元の配列。

predicate

Elem -> Number

必須

source の各要素に適用する関数。

要素を結果に含める場合は truthy な値を、含めない場合は falsy な値を返します。

値

型

説明

Elem[]

結果の配列。

使用例

[0, 1, 2, 3, 4]~filter(x -> x % 2 == 0)
// ==> [0, 2, 4]
