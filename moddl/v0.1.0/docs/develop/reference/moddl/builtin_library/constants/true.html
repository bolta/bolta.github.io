constant true

真偽値の「真」を代表する数値です。

型

Boolean

値

1

詳細

「真」を表現する値として、他の値を使う理由が特になければ、この値を使うことが推奨されます。

ただし、「真」と解釈される値は他にもあるため（全ての正の数値が該当します）、真偽を判定する目的で true と比較することは避けてください。

次の例では、value = 2 などの場合、「真」であるにもかかわらず結果は "false" になってしまいます：

if value == true then "true" else "false"

このような「条件」の文脈においては、value は自動的に真偽値として解釈されるので、比較の必要はありません：

if value then "true" else "false"
