constant false

真偽値の「偽」を代表する数値です。

型

Boolean

値

-1

詳細

「偽」を表現する値として、他の値を使う理由が特になければ、この値を使うことが推奨されます。

ただし、「偽」と解釈される値は他にもあるため（全ての負の数値と 0 が該当します）、真偽を判定する目的で false と比較することは避けてください。

次の例では、value = 0 などの場合、「偽」であるにもかかわらず結果は "true" になってしまいます：

if value == false then "false" else "true"

「偽」であることを判定するには ! 演算子を使ってください：

if ! value then "false" else "true"
