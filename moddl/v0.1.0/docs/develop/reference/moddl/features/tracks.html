トラックについて

トラックとは、音を出したり加工したりする役割を担う単位です。

トラックは MML によって独立に操作することができます。

トラックには instrument トラック、effect トラック、groove トラックの 3 種類があります。それぞれの役割は以下の通りです：

  • instrument トラック
    • 音を生成する
  • effect トラック
    • 生成された音を加工する
  • groove トラック
    • シーケンサの tick の発生タイミングを変化させる

instrument トラックは、周波数の時系列を入力に取り、波形を出力します。入力となる周波数の時系列はシーケンサによって制御されます。

effect トラックは、波形を入力に取り、それを加工した波形を出力します。入力される波形は他の instrument または effect トラックの出力です。複数の出力を受け取ることもできます。

groove トラックは、tick タイマーの出力を入力に取り、それに変調を加えたものを出力します。変調を加えた tick タイマーを使うのは、他のトラックを操作するシーケンサです。

それぞれのトラックを定義するには、@instrument, @effect, @groove の各構築文を使います。
