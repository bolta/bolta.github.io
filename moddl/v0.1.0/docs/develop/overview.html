ModDL/MML の概要

ModDL/MML は、音楽のための 2 つのプログラミング言語である ModDL と MML を合わせたものです。これらの言語の概要をここで説明します。

ModDL

ModDL は、音楽を演奏するためのモジュールを記述する DSL（ドメイン特化言語 = 特定用途の言語）です。

モジュールという単語はプログラミングの世界でも出てきますが、ここで言うモジュールはむしろモジュラーシンセサイザの用語でいうモジュールに似たもので、音などの信号を生成したり変換したりする機構のことを指します。広い意味での「楽器」に相当するものです。

ModDL はコードによってモジュールの構造を表現します。モジュールの内部構造を定義する、あるいはモジュール同士を接続する、といった、モジュールの設計に関すること全般を、簡潔に、かつ柔軟に表現することができます。

ModDL の名前は Module Definition Language（モジュール定義言語）または Module Description Language（モジュール記述言語）に由来します。読み方は “model” と同じです。

ModDL はモジュールを定義する役割のほかに、次に述べる MML のホスト言語としての役割も持っています。

MML

ModDL に組み込まれた形で、もう一つの DSL、MML（Music Macro Language）が存在します。

ModDL が「楽器」を定義するのに対して、MML は「楽譜」を定義します。ModDL で定義された「楽器」を使って、MML で定義された「楽譜」を演奏した結果が「音楽」となります。

伝統的には、MML は 1980 ～ 90 年代のアマチュアコンピュータ音楽シーンを中心に愛用された言語です。MML は基本的に特定機種のハードウェア（コンピュータや音源チップ）を演奏するために提供された言語であり、標準規格などは今に至るまで存在しないため、多くの方言がありました。

ModDL にバンドルされた MML も、そのような方言の新たに加わった一つといえますが、ModDL の MML は ModDL によるモジュール設計の自由度の高さを反映して、特定のモジュール構成を仮定せず、どのような構成であってもその機能を不足なく活用する能力を持っているのが特徴です。

本文書群において単に MML と言うときは、この ModDL 版の MML を指します。

ModDL/MML

ModDL とそれにバンドルされた ModDL 版 MML を併せて ModDL/MML と表記することがあります。

ModDL, MML とも DSL の一種ですが、プログラミング言語としての特徴を顕著に備えています。ModDL/MML は音楽のためのプログラミング環境であり、「プログラマのメンタルモデルで音楽をする」ためのツールと言うこともできます。

「ModDL/MML の処理系」の意味で、小文字の moddl を使う場合があります。これは処理系のコマンド名 moddl（ファイル名 moddl.exe）に由来します。
