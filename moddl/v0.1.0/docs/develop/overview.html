ModDL/MML の概要

ModDL

ModDL は、音楽を演奏するためのモジュールを記述する DSL（ドメイン特化言語 = 特定用途の言語）です。

ここでいうモジュールとは、（プログラミングでもモジュールという単語は出てきますが、それよりはむしろ）モジュラーシンセサイザでいうモジュールと似たものであり、音などの信号を生成したり変換したりする機構のことです。いわば、広い意味での「楽器」に相当します。

ModDL のコードを記述することで、モジュールを自由に設計したり、またモジュール同士をつなぎ合わせたりすることができます。

ModDL の名前は Module Definition Language（モジュール定義言語）または Module Description Language（モジュール記述言語）に由来し、発音は “model” と同じです。

MML

ModDL のコードに含まれる形で、さらに別の DSL が存在します。それは MML（Music Macro Language）です。

先述の ModDL が「楽器」を定義するものであったのに対して、MML は「楽譜」を定義します。ModDL で作られた楽器を操って「音楽」を表現するための情報を記述するのが MML です。

本来、MML は 1980 ～ 90 年代のアマチュアコンピュータ音楽シーンを中心に用いられた言語です。特定のハードウェア（コンピュータあるいは音源チップ）に強く結合した言語であったことを一因として、多種多様の方言がありました。ModDL にバンドルされた MML もそのような方言の新しい一つといえますが、ModDL の MML の特徴は、ModDL によるモジュール設計の自由度の高さを反映して、特定の音源形態を仮定せず、どのようなモジュール構成であってもその機能を不足なく活用する能力を持っていることです。

本文書群において単に MML と言うときは、この ModDL 版の MML を指します。

ModDL/MML

ModDL とそれにバンドルされた ModDL 版 MML を併せて ModDL/MML と表記することがあります。

ModDL, MML とも DSL の一種ですが、プログラミング言語としての特徴を顕著に備えています。ModDL/MML は音楽のためのプログラミング環境であり、「プログラマのメンタルモデルで音楽をする」ためのツールと言うこともできます。

「ModDL/MML の処理系」の意味で、小文字の moddl を使う場合があります。これは処理系のコマンド名 moddl（ファイル名 moddl.exe）に由来します。
