function flatMap

配列の各要素に関数を適用して任意個の値を含む配列に変換し、その配列を全て連結した 1 次元の配列を得ます。

型定義

mapper

SrcElem -> DestElem[]

必須

source の各要素に適用し、結果の配列の要素となる値の配列を生成する関数。

値

型

説明

DestElem[]

結果の配列。

使用例

[0, 1, 2, 3, 4]~flatMap(x -> [x, x ^ 2, x ^ 3])
// ==> [0, 0, 0, 1, 1, 1, 2, 4, 8, 3, 9, 27, 4, 16, 64]

// 下記と同じ
[0, 1, 2, 3, 4]~map(x -> [x, x ^ 2, x ^ 3])~flat()

詳細

この関数は、map 関数の結果に flat 関数を適用するのと同じです。
