function sum

配列の全ての要素の総和を求めます。

型定義

名前

型定義

Elem

*
+ 演算子の仕様に従います。

Result

*
+ 演算子の仕様に従います。

引数

名前

型

必須/省略時

説明

source

Elem[]

必須

集計元の配列。

値

型

説明

Result

source の全ての要素の総和。

使用例

[0, 1, 2, 3, 4, 5, 6]~sum()
// ==> 21

// 加算合成で矩形波を生成する（配列は任意個の連番とする）
[0, 1, 2, 3, 4, 5, 6]~map(x -> let k = 2 * x + 1; =f=> (k * f) | sineOsc / k)~sum()

// 文字列を連結する
["foo", "bar", "baz"]~sum()
// ==> "foobarbaz"

詳細

この関数は、配列の全ての要素の間に + 演算子を挿入した式と同じ結果を得ます。

したがって、例に挙げたように、+ 演算子が対応している型の値の配列であれば同様に扱うことができます。
